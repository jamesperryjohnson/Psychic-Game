SRE Mt Fuji - PingCloud

SRE Mt Fuji Interview Process Submission - PingCloud

This is a take-home programming assignment that I completed as part of the Ping Identity interview process. This consists of a coding problem and a two-part DevOps problem. This package consists of the solution for the coding problem and the first part of the DevOps problem.

Getting Started

These instructions will cover usage information for the Docker container.

Prerequisities

In order to run this container you'll need Docker installed.

  • Windows
  • OS X
  • Linux

Additional Required Tools

  • Git
  • Python 3.7
  • Pipenv

Usage

Building the Docker image for this project

On your local machine, clone this repo:

git clone --recursive https://github.com/jamesperryjohnson/sre-fuji-submission-pingcloud.git
                cd sre-fuji-submission-pingcloud

Then build a Docker image for this project using the following command:

docker build -t my-image-name -f Dockerfile .

where my-image-name is the name you want to give your created Docker image.

Running the Docker image for this project

After the Docker image has been created for this project, you can run it using either of the following commands:

  • Run the main project script from command line:
      docker run my-image-name
  • Run project tests from command line:
      docker run my-image-name pytest

To rebuild the Docker image after changes are made, repeat the steps above.
